# StorageCommand

Self-service storage unit management app with a 3D interactive facility map.

## Features
- 3D isometric facility map with pan/zoom/rotate (touch + mouse)
- Real-time unit status (Available, Occupied, Reserved, Maintenance, Past Due)
- Customer self-service rental flow (unit select → info → payment → e-sign → access codes)
- Owner dashboard with tenant info and revenue overview
- Facility setup & configuration screen

## Stack
- React + Vite
- Canvas-based 3D renderer (no Three.js dependency)

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
